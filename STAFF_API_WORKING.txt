===========================================
STAFF API (Separate Section)
===========================================

Base URL: /api/staff

✅ Authentication:
   POST /api/staff/login - Staff login (returns token)
   POST /api/staff/logout - Logout
   GET  /api/staff/profile - Get profile
   POST /api/staff/refresh-token - Refresh token

✅ Dashboard:
   GET /api/staff/dashboard - Full dashboard with stats
   GET /api/staff/performance - Performance metrics

✅ Tasks:
   GET  /api/staff/tasks - List all tasks
   GET  /api/staff/tasks/{uuid} - Task details
   POST /api/staff/tasks/{uuid}/start - Start task
   POST /api/staff/tasks/{uuid}/complete - Complete task
   POST /api/staff/tasks/{uuid}/upload-proof - Upload proof photos

✅ Attendance:
   GET  /api/staff/attendance - History
   GET  /api/staff/attendance/today - Today's record
   POST /api/staff/attendance/check-in - Check in
   POST /api/staff/attendance/check-out - Check out

✅ Leave Requests:
   GET  /api/staff/leave-requests - List requests
   POST /api/staff/leave-requests - Submit request

===========================================
OWNER API (Manage Staff)
===========================================

Base URL: /api/owner

✅ Staff Management:
   GET    /api/owner/staff - List all staff (25 staff found!)
   POST   /api/owner/staff - Create staff
   GET    /api/owner/staff/{uuid} - Staff details
   PUT    /api/owner/staff/{uuid} - Update staff
   DELETE /api/owner/staff/{uuid} - Delete staff

✅ Hierarchy & Departments:
   GET /api/owner/properties/{id}/hierarchy - Staff hierarchy
   GET /api/owner/departments - List departments
